# World of Sea Battle - Interactive Map Dashboard 🏴‍☠️

A comprehensive, real-time interactive map dashboard for the game **World of Sea Battle**. This tool allows players to track wind conditions, discover the most profitable trade routes, manage port information, and plan their naval strategies with specialized drawing tools.

![Dashboard Preview](assets/img/preview.png) *(Note: Placeholder for actual preview image)*

## 🌟 Features

- **🌬️ Real-time Wind Tracking**: Precision wind angle adjustments and synchronization with game time.
- **📈 Advanced Trade Suggestions**: Calculate the most profitable trade routes based on gold per distance and cargo weight.
- **🏭 Production Site Filtering**: Easy-to-use filters for resources and production areas.
- **⚓ Port Management**: Track port owners, clans, and active war timers in real-time.
- **💎 Regional Resource Tracking**: Live list of regional points and resources with map navigation.
- **🎨 Strategic Drawing Tools**: Custom tools for route planning, area marking, and tactical icons.
- **🌍 Multi-language Support**: Full localization for Turkish, English, Russian, Spanish, and Chinese.

## 🛠️ Technology Stack

- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Map Engine**: Leaflet.js
- **Styling**: Tailwind CSS (Headless UI components) / Vanilla CSS
- **Data Persistence**: LocalStorage for UI settings and port configurations.
- **API Integration**: Real-time global hit counter via CounterAPI.

## 🚀 Getting Started

1. **Clone the repository**:
   ```bash
   git clone https://github.com/enesatacdev/wosb-interactive-map.git
   ```
2. **Open the project**:
   Simply open `index.html` in your favorite web browser (Chrome, Firefox, Edge, etc.).
   *Note: For the best experience (and to avoid CORS issues locally), using a local server like "Live Server" for VS Code is recommended.*
